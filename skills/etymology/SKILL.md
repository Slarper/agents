name: etymology
description: 拆解英文单词的词源（前缀/词根/后缀），并给出同一词干的同源词，用中文回答
---

## When to Use This Skill

当用户给出一个单词时，使用本技能。典型触发句式：

- 用户直接给出一个英文单词（例如："pandemic"）
- 用户说 "帮我拆解一下这个词" 或类似表达
- 用户以命令格式调用：`/skills <word>`，例如 `/skills hello`

## Command Invocation

当用户通过 `/skills <word>` 调用时，忽略命令本身（`/skills`），将后面的 `<word>` 作为待处理单词。若有多个单词传入，只处理第一个单词。

## Skill Behavior

对于用户给的每一个单词，按以下步骤处理：

### Step 1: 确定词源

查阅权威词源资源，确定该词的**词根（root）**来源（通常来自拉丁语或希腊语）。

### Step 2: 输出格式

全部用**中文**回答。格式如下：

```
## [单词]

**词源拆解：**
[前缀/词根/后缀] → 含义 → 组合含义

**同源词（同一词干）：**
- [单词1] - [含义]
- [单词2] - [含义]
- [单词3] - [含义]
- [单词4] - [含义]
- [单词5] - [含义]
```

### 规则说明

1. **词源拆解**：拆解前缀（prefix）、词根（root）、后缀（suffix），分别说明其来源语言和含义，然后说明组合后的整体含义。
2. **同源词**：选取与词根相同的高频英语单词，**排除**仅共享常见前后缀的词（如 -tion, -ly, un-, re- 等），最多5个。
3. **语言**：回答主体必须全部使用中文（单词本身和必要的拉丁/希腊术语除外）。

## Examples

### 直接输入单词

用户输入：`pandemic`

模型回复：

```
## pandemic

**词源拆解：**
pan-（希腊前缀，意为"全部、所有"）+ demos（希腊语，意为"人民、民众"）+ -ic（形容词后缀）→ "波及全体人民的" → 大流行病

**同源词（同一词干）：**
- democracy - 民主（demos "人民" + kratos "统治"）
- demography - 人口统计学（demos "人民" + graphia "描写"）
- endemic - 地方性的（en "在...内" + demos "人民"）
- epidemic - 流行病（epi "在...之间" + demos "人民"）
- demagogue - 煽动者（demos "人民" + agogos "引导"）
```

### 通过命令调用

用户输入：`/skills hello`

模型回复：

```
## hello

**词源拆解：**
古高地德语 hala（招呼语，意为"来这儿"）→ 演变自 hola，用于引起注意 → 英语中的问候语

**同源词（同一词干）：**
- hallo - 喂（同源变体）
- holla - 喂（变体形式）
- hey - 嘿（类似感叹词）
- hooray - 好哇（感叹词，同源不确定）
- lo - 看啊（古英语感叹词，部分同源）
```
